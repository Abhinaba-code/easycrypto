# **App Name**: AstraPulse: Crypto Arcade

## Core Features:

- Local Authentication: Register and login using locally stored credentials with password hashing.
- Realtime Crypto Data Feed: Fetch and display live market data from CoinGecko API and optionally Binance WebSocket, cached for performance.
- Recommendation Engine: Generative AI powered tool providing crypto recommendations based on user profile, investment horizon, and experience using volume spike, momentum and volatility.
- Portfolio Simulator: Simulate a crypto portfolio with fake currency, track trades, and calculate profit/loss, stored in LocalStorage.
- Crypto Games: Play games such as Price Prediction, Dice, and Coin Flip with fake coins. Outcomes are verifiable client-side.
- In-App Notifications: Receive in-app notifications for signals, achievements, and P/L moves, with optional browser notifications.
- Data Management: Allow users to export and import portfolio and game data using a JSON file.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), evoking trust and innovation.
- Background color: Very dark grey (#212121), providing contrast.
- Accent color: Electric blue (#3F51B5), for highlights and calls to action.
- Body and headline font: 'Space Grotesk' sans-serif, for a modern, technical feel; 'Inter' sans-serif for longer text blocks.
- Code font: 'Source Code Pro' monospaced, for displaying code snippets.
- Use clean, minimalist icons representing crypto and gaming concepts.
- Incorporate framer-motion for micro-interactions on buttons and cards, staggered entrance animations, and price pulse animations.
